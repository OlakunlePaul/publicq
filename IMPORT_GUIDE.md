# Word Import Formatting Guide

Follow these simple rules to ensure 100% accurate imports of mathematical and non-mathematical questions.

## 1. Questions MUST Start with a Number
Every new question should start with its number followed by a dot or parenthesis.
*   **Correct**: `1. What is the value of x?`
*   **Correct**: `1) Solve the equation.`
*   **Incorrect**: `Solve this: 1. What is x?` (The question text must come after the number).

## 2. Options (A-E)
Options must use letters `a)` through `e)` and should ideally be on the same line or immediately following the question.
*   **Correct**: `a) 10 b) 20 c) 30* d) 40` (The `*` marks the correct answer).
*   **Correct**: 
    ```
    a) Option A
    b) Option B
    ```

## 3. Mathematical Equations
To prevent equations from being split into separate questions, ensure the entire equation is followed by the options.
*   **Pro Tip**: If a complex equation is split across multiple lines in Word, our "Smart Merge" will now keep them together as long as you haven't started the options yet.

## 4. Tables and Images (Content Context)
If you have a table or graph that applies to one or more questions, place it **immediately before** the first question it refers to.
*   **Format**:
    ```
    [Table/Image here]
    Use this table to answer questions 8 and 9.
    
    8. What is the value of...
    a) ... b) ...
    
    9. Based on the same table...
    a) ... b) ...
    ```

## 5. Marking Correct Answers
Add an asterisk `*` or `**` to the end of the correct option text, or write `(correct)` next to it.
*   **Example**: `c) 42*` or `c) 42 (correct)`
