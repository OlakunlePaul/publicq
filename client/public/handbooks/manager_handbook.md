# 👨‍💼 Manager Handbook: System Setup & Oversight

Welcome to the Manager Dashboard! Your role is to build the "Blueprint" of the school and oversee the "Registry" and "Academic Vault."

---

## 🏛️ Phase 1: The Blueprint (School Structure)
Before teachers can set exams, you must define the school's organization.

1.  **Sessions**: Go to **School Structure** > **Sessions**. Create a new session (e.g., "2025/2026 Academic Session").
2.  **Terms**: Under your new session, add **Terms** (e.g., "First Term", "Second Term").
3.  **Classes & Subjects**: Define the levels (e.g., "Primary 1") and the subjects taught in each (e.g., "Mathematics").
    *   *Tip: Link each Class to a "Grading Schema" if you want automated letter grades (A, B, C).*

---

## 📋 Phase 2: The School Registry (Users & Admissions)
Populate the school and manage student identities.

1.  **Registration Format**: Go to **Registration Format**. Set how IDs are issued (e.g., `DISC/{YYYY}/{0000}`). 
    *   `{YYYY}` is the current year.
    *   `{0000}` is a sequential number that increments automatically.
2.  **Adding Users**: Go to **Users**.
    *   **Students**: Click "Create New User," toggle **Create as Student**, and leave the "Admission Number" blank for auto-issuance.
    *   **Staff/Teachers**: Create users and assign the appropriate **Role** (Teacher, Manager, or Admin).
    *   **Parents**: Link parents to their children so they can view **Report Cards**.

---

## 📂 Phase 3: The Academic Vault (Report Cards & Approvals)
Monitor performance and finalize records.

1.  **Report Cards**: Go to **Report Cards**.
2.  **Broadsheet View**: Open the "Broadsheet" for any class. This is your master ledger where you can see every student's scores across all subjects.
3.  **Audit**: Ensure teachers have synced all online scores or entered manual marks (e.g., Practicals/CA).
4.  **Oversight**: Review individual **Report Cards** before they are released to parents and students.
